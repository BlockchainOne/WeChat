## 讨论

**btc wallet**

* 现在 go 语言的 btcwallet 是不是 不能用 bitcoin core 钱包同步的区块当节点
* github 上的 btcwallet 这么说的  另一种与 BTCD 和 bitcoin core 兼容的 SPV 模式计划在未来发布
* 比特币的，go 语言开发钱包 是不是只能使用 go 语言的 btcwallet 和 btcd
* python，java 都行
