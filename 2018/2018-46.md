## 讨论

**行情数据 API**

* 内部系统用，访问量就几十个人
* 一般从各大交易所拿，然后可以自己取平均什么的
* CoinMarketCap，免费的 API 够用了
* 别的媒体也是要从交易所拿的
* 交易所的 api 好像有很多问题，搞不明白
* 金色我记得用的也是 coinmarketcap
